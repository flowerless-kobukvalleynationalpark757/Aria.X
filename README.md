## 💎 Features of Aria.X

Aria.X organizes your work. It tracks every file you download. If your internet connection stops, the program pauses the transfer. When your connection returns, the program starts the download again from the same spot. This prevents data loss. You can group files by category to keep your computer clean. The interface shows the speed of each transfer. You see the estimated time remaining for every active task. 

## 🛠️ System requirements for Windows

Your computer needs Windows 10 or Windows 11 to run this software. Ensure you have at least 200 megabytes of free space on your hard drive. A stable internet connection helps the program reach top speeds. If you have older hardware, the software will still function, but it may take more time to load the interface. 

## 🖱️ Using the main interface

The main window shows your active downloads in a clear list. To add a new file, copy the web link of the file you want. Click the large plus button inside the Aria.X window. Paste the link into the box and press the button labeled Start. The program takes over from there. You can stop or cancel any task by clicking the icons next to the file name. 

## 🧩 Managing your library

The library tab keeps a history of your past transfers. You can sort these files by date, size, or name. Right-click any file in the history to open the folder where the file lives. This saves time when you lose track of where a download ended up. 

## 🛡️ Privacy and safety

Aria.X keeps your information on your own machine. We do not track the files you download. The software only connects to the web servers needed to complete your tasks. It does not share your data with other parties. Your download history stays local to your computer.
